This project uses open-meteo API and provide real-time weather information using city's name or zipcode. Users can input the name of a city or zipcode, and the application will fetch and display the current weather conditions, including temperature, humidity, and wind speed and shows icons like sunny, rainy day, snow, shower, mist or cloud. And it has a default picture. When city or zipcode doesn't exist in API, the page show information" not found". I used addition API Opencagedata.com to convert city name to geolocation. Also I used Zippopotam.us for converting zipcode to coordinate.
